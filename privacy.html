Privacy Policy

Last updated: April 8, 2026

Overview

CINEMA SLIDE is a browser-based video creation tool. Your privacy is fundamental to how this application is built. All photo and video processing happens entirely within your browser. Your photos and videos never leave your device.

Data We Do NOT Collect

  • We do not collect, upload, store, or transmit your photos, videos, or audio files
  • We do not require account creation or login
  • We do not use cookies for tracking
  • We do not use third-party analytics services
  • We do not share any data with third parties

How the Application Works

CINEMA SLIDE processes all images and generates videos entirely in your web browser using the WebCodecs API. Your photos are loaded into browser memory, rendered to a canvas, encoded into MP4 format, and made available for download — all without any server communication.

Local Storage

The application uses your browser's localStorage to save:

  • Language preference (English or Japanese)
  • User profile settings (default duration, resolution, FPS, and optional display name for credits)
  • Anonymous usage events (e.g., "export started," "export completed") stored locally for diagnostic purposes only — never transmitted externally. Capped at 200 events.

You can clear this data at any time by clearing your browser's site data or using the "Reset" button in the profile settings.

External Resources

The application loads the following external resources:

  • Google Fonts (fonts.googleapis.com) — Playfair Display and IBM Plex Mono typefaces. Google's Font API privacy policy applies.
  • mp4-muxer library (cdn.jsdelivr.net or esm.sh) — An open-source JavaScript library used for MP4 file assembly. No user data is sent to these CDNs.

Children's Privacy

The application does not knowingly collect data from children under 13. Since no personal data is collected from any user, no special provisions are required.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected on this page with an updated date.

Contact

If you have questions about this Privacy Policy, please open an issue on our GitHub repository.
